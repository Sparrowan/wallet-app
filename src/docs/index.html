Wallet Management System Documentation

Introduction

The Wallet Management System is a web application designed to facilitate the management of users' wallets, including functionalities such as creating wallets, listing transactions, crediting and debiting wallets, and viewing wallet details.

Technical Decisions

  • Backend Framework: Node.js with Express.js
  • Database: MongoDB with Mongoose
  • API Design: RESTful Architecture
  • Validation: Yup Schema Validation
  • Frontend Styling: Tailwind CSS
